This section gives syntax definitions for the derived expression types in terms of the primitive expression types (literal, variable, call, `=13`lambda, `=13`if, and `=13`set!), except for `=13`quasiquote.

The following `=13`letrec macro uses the symbol `=13`<undefined> in place of an expression which returns something that when stored in a location makes it an error to try to obtain the value stored in the location. (No such expression is defined in Scheme.) A trick is used to generate the temporary names needed to avoid specifying the order in which the values are evaluated. This could also be accomplished by using an auxiliary macro.

Here is a possible implementation of `=13`delay, `=13`force and `=13` delay-force. We define the expression


unspecified error makeother^`=13` 


          gobblecr(delay-force expression)

to have the same meaning as the procedure call


unspecified error makeother^`=13` 


          gobblecr(make-promise #f (lambda () expression))

as follows


unspecified error makeother^`=13` 


          gobblecr(define-syntax delay-force  (syntax-rules ()    ((delay-force expression)      (make-promise #f (lambda () expression)))))

and we define the expression


unspecified error makeother^`=13` 


          gobblecr(delay expression)

to have the same meaning as:


unspecified error makeother^`=13` 


          gobblecr(delay-force (make-promise #t expression))

as follows


unspecified error makeother^`=13` 


          gobblecr(define-syntax delay  (syntax-rules ()    ((delay expression)     (delay-force (make-promise #t expression)))))

where `=13`make-promise is defined as follows:


unspecified error makeother^`=13` 


          gobblecr(define make-promise  (lambda (done? proc)    (list (cons done? proc))))

Finally, we define `=13`force to call the procedure expressions in promises iteratively using a trampoline technique following [38] until a non-lazy result (i.e. a value created by `=13` delay instead of `=13`delay-force) is returned, as follows:


unspecified error makeother^`=13` 


          gobblecr(define (force promise)  (if (promise-done? promise)      (promise-value promise)      (let ((promise* ((promise-value promise))))        (unless (promise-done? promise)          (promise-update! promise* promise))        (force promise))))

with the following promise accessors:


unspecified error makeother^`=13` 


          gobblecr(define promise-done?  (lambda (x) (car (car x))))(define promise-value  (lambda (x) (cdr (car x))))(define promise-update!  (lambda (new old)    (set-car! (car old) (promise-done? new))    (set-cdr! (car old) (promise-value new))    (set-car! new (car old))))
