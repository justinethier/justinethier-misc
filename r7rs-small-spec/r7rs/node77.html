Vectors

Vectors are heterogeneous structures whose elements are indexed by integers. A vector typically occupies less space than a list of the same length, and the average time needed to access a randomly chosen element is typically less for the vector than for the list.

The length of a vector is the number of elements that it contains. This number is a non-negative integer that is fixed when the vector is created. The valid indexesindexfile"valid indexes" "rm" aux of a vector are the exact non-negative integers less than the length of the vector. The first element in a vector is indexed by zero, and the last element is indexed by one less than the length of the vector.

Vectors are written using the notation #(obj ). For example, a vector of length 3 containing the number zero in element 0, the list `=13`(2 2 2 2) in element 1, and the string `=13`"Anna" in element 2 can be written as follows:


unspecified error makeother^`=13` 


          gobblecr#(0 (2 2 2 2) "Anna")

Vector constants are self-evaluating, so they do not need to be quoted in programs.





`=13`=0.5em(indexfile"vector?" "tt" main vector?obj) procedure

Returns #t if obj is a vector; otherwise returns #f.





`=13`=0.5em(indexfile"make-vector" "tt" main make-vectork) procedure
`=13`=0.5em(make-vectork fill) procedure

Returns a newly allocated vector of k elements. If a second argument is given, then each element is initialized to fill. Otherwise the initial contents of each element is unspecified.






`=13`=0.5em(indexfile"vector" "tt" main vectorobj ) procedure

Returns a newly allocated vector whose elements contain the given arguments. It is analogous to `=13`list.


unspecified error makeother^`=13` 


          gobblecr(vector 'a 'b 'c)               #(a b c)





`=13`=0.5em(indexfile"vector-length" "tt" main vector-lengthvector) procedure

Returns the number of elements in vector as an exact integer.





`=13`=0.5em(indexfile"vector-ref" "tt" main vector-refvector k) procedure

It is an error if is not a valid index of vector.

The `=13`vector-ref procedure returns the contents of element of vector.


unspecified error makeother^`=13` 


          gobblecr(vector-ref '#(1 1 2 3 5 8 13 21)            5)  8(vector-ref '#(1 1 2 3 5 8 13 21)            (exact             (round (* 2 (acos -1))))) 13





`=13`=0.5em(indexfile"vector-set!" "tt" main vector-set!vector k obj) procedure

It is an error if is not a valid index of vector.

The `=13`vector-set! procedure stores obj in element of vector.
unspecified error makeother^`=13` 


          gobblecr(let ((vec (vector 0 '(2 2 2 2) "Anna")))  (vector-set! vec 1 '("Sue" "Sue"))  vec)      #(0 ("Sue" "Sue") "Anna")


(vector-set! '#(0 1 2) 1 "doe")  ; constant vector





`=13`=0.5em(indexfile"vector->list" "tt" main vector->listvector) procedure
`=13`=0.5em(vector->listvector start) procedure
`=13`=0.5em(vector->listvector start end) procedure
`=13`=0.5em(indexfile"list->vector" "tt" main list->vectorlist) procedure

The `=13`vector->list procedure returns a newly allocated list of the objects contained in the elements of vector between start and end. The `=13`list->vector procedure returns a newly created vector initialized to the elements of the list list.

In both procedures, order is preserved.


unspecified error makeother^`=13` 


          gobblecr(vector->list '#(dah dah didah))  (dah dah didah)(vector->list '#(dah dah didah) 1 2) (dah)(list->vector '(dididit dah))   #(dididit dah)





`=13`=0.5em(indexfile"vector->string" "tt" main vector->stringvector) procedure
`=13`=0.5em(vector->stringvector start) procedure
`=13`=0.5em(vector->stringvector start end) procedure
`=13`=0.5em(indexfile"string->vector" "tt" main string->vectorstring) procedure
`=13`=0.5em(string->vectorstring start) procedure
`=13`=0.5em(string->vectorstring start end) procedure

It is an error if any element of vector between start and end is not a character.

The `=13`vector->string procedure returns a newly allocated string of the objects contained in the elements of vector between start and end. The `=13`string->vector procedure returns a newly created vector initialized to the elements of the string string between start and end.

In both procedures, order is preserved.


unspecified error makeother^`=13` 


          gobblecr(string->vector "ABC")  #(#\A #\B #\C)(vector->string  #(#\1 #\2 #\3) "123"





`=13`=0.5em(indexfile"vector-copy" "tt" main vector-copyvector) procedure
`=13`=0.5em(vector-copyvector start) procedure
`=13`=0.5em(vector-copyvector start end) procedure

Returns a newly allocated copy of the elements of the given vector between start and end. The elements of the new vector are the same (in the sense of `=13`eqv?) as the elements of the old.


unspecified error makeother^`=13` 


          gobblecr(define a #(1 8 2 8)) ; a may be immutable(define b (vector-copy a))(vector-set! b 0 3)   ; b is mutableb #(3 8 2 8)(define c (vector-copy b 1 3))c #(8 2)





`=13`=0.5em(indexfile"vector-copy!" "tt" main vector-copy!to at from) procedure
`=13`=0.5em(vector-copy!to at from start) procedure
`=13`=0.5em(vector-copy!to at from start end) procedure

It is an error if at is less than zero or greater than the length of to. It is also an error if `=13`(- (vector-length to) at) is less than `=13`(- end start).

Copies the elements of vector from between start and end to vector to, starting at at. The order in which elements are copied is unspecified, except that if the source and destination overlap, copying takes place as if the source is first copied into a temporary vector and then into the destination. This can be achieved without allocating storage by making sure to copy in the correct direction in such circumstances.


unspecified error makeother^`=13` 


          gobblecr(define a (vector 1 2 3 4 5))(define b (vector 10 20 30 40 50))(vector-copy! b 1 a 0 2)b #(10 1 2 40 50)





`=13`=0.5em(indexfile"vector-append" "tt" main vector-appendvector ) procedure

Returns a newly allocated vector whose elements are the concatenation of the elements of the given vectors.


unspecified error makeother^`=13` 


          gobblecr(vector-append #(a b c) #(d e f)) #(a b c d e f)
