Install using Cabal

The easiest way to install husk is via Cabal: 

cabal install husk-scheme

Build from Source

Alternatively, husk may be built from source. The first thing you need to do is download the latest version of husk from here. Once husk has been downloaded, the following commands may be used to build, install, and optionally test the husk executable and libraries: 

make clean

# Install husk to your HOME directory
make dist

# An optional step to run all the built-in unit tests
make test
