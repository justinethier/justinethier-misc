Install using Cabal

The easiest way to install husk is via Cabal: {% highlight console %} cabal install husk-scheme {% endhighlight %}

Build from Source

Alternatively, husk may be built from source. The first thing you need to do is download the latest version of husk from here.

Before building husk, all of the required dependencies must be available. On Fedora these dependencies are:

  • ghc
  • ghc-doc
  • ghc-parsec-devel
  • ghc-haskeline-devel
  • ghc-ghc-paths-devel
  • ghc-ghc-devel
The same packages are necessary regardless of your platform, but the names may differ.

Once husk has been downloaded and dependencies have been met, the following commands may be used to build, install, and optionally test the husk executable and libraries: {% highlight console %} make clean # Install husk to your HOME directory make dist # An optional step to run all the built-in unit tests make test {% endhighlight %}
